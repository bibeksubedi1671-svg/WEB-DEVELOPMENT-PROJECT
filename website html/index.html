HTML (HyperText Markup Language)

HTML is the markup language used to define the structure of web content. It provides elements such as headings, paragraphs, lists, tables, images, links, and forms. Modern HTML emphasizes semantic elements, accessibility, and mobile-friendly features.

Example of HTML

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

CSS (Cascading Style Sheets)

CSS is used to style HTML content by changing colors, layouts, fonts, and responsiveness.

body {
  background: beige;
  color: black;
  font-family: Arial, sans-serif;
}
h1 {
  color: blue;
  text-align: center;
}

Bootstrap Framework

Bootstrap provides prebuilt CSS and JavaScript components that make it easy to build responsive websites.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css">

<div class="container text-center">
  <h1 class="text-primary">Hello Bootstrap</h1>
  <button class="btn btn-success">Click Me</button>
</div>

JavaScript

JavaScript makes web pages interactive. It can change content, validate forms, and fetch data without reloading the page.

<button onclick="changeText()">Click Me</button>
<p id="demo">Hello</p>

<script>
  function changeText() {
    document.getElementById("demo").innerHTML = "You clicked the button!";
  }
</script>

Full Stack Development

Full Stack combines front-end (HTML, CSS, JS) and back-end (server, APIs, databases) to build complete applications.

// Backend: Node + Express
const express = require('express');
const app = express();
app.get('/hello', (req,res) => res.send('Hello from server!'));
app.listen(3000);

// Frontend: HTML + fetch()
<button onclick="loadData()">Fetch</button>
<p id="out"></p>
<script>
async function loadData() {
  let res = await fetch('http://localhost:3000/hello');
  let text = await res.text();
  document.getElementById('out').innerText = text;
}
</script>

Summary: HTML structures your content, CSS makes it beautiful, Bootstrap speeds up styling, JavaScript adds interactivity, and Full Stack ties it all together with backend logic and databases.
