Full Stack Development

Full Stack Development combines both the front-end and back-end of applications. A full stack developer builds user interfaces (HTML, CSS, JS) and connects them with servers, databases, and APIs.

Example

// Backend: Node + Express
const express = require('express');
const app = express();
app.get('/hello', (req,res) => res.send('Hello from server!'));
app.listen(3000);

// Frontend fetch
<button onclick="loadData()">Fetch</button>
<p id="output"></p>
<script>
async function loadData() {
  let res = await fetch('http://localhost:3000/hello');
  document.getElementById('output').innerText = await res.text();
}
</script>
