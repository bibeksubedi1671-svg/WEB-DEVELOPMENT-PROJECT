HTML (HyperText Markup Language)

HTML structures a webpage with semantic tags. It provides headings, paragraphs, links, images, tables, and forms.

Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
